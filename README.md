# M&T Services — Website

Marketing site for **M&T Services** — modern website design & custom AI assistants for small businesses.

- **Stack:** plain HTML, CSS, and JavaScript (no build step)
- **Sections:** Hero, Services, Pricing, Process, Why Us, Contact + live AI assistant demo
- **Contact form:** wired for [Netlify Forms](https://docs.netlify.com/forms/setup/) (works automatically once deployed to Netlify)

## Local preview
```bash
python -m http.server 8123
# then open http://localhost:8123
```

## Deploy
Hosted on Netlify. Pushing to the `main` branch (or running `netlify deploy --prod`) publishes the site.
